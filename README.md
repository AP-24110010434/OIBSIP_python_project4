## Objective

To develop a weather application that retrieves and displays real-time weather information using an API.

## Tools Used

- Python
- Tkinter
- Requests
- OpenWeatherMap API

## Steps Performed

1. Designed the GUI using Tkinter.
2. Integrated the OpenWeatherMap API.
3. Implemented city-based weather search.
4. Retrieved weather data in JSON format.
5. Parsed and displayed temperature, humidity, and wind speed.
6. Stored search history.
7. Implemented exception handling and input validation.

## Outcome

The application successfully fetches and displays real-time weather information for user-specified cities through an interactive GUI.
